# instructions

do a git clone on this repo on your command line

once its installed, run these commands

```
yarn
yarn dev
```


# building help

if you want to build this, use claude to help you with the code

the frontend is built with next js, tailwind, typescript https://nextjs.org/docs/app/building-your-application/routing

you can setup a database to store the book and profile data with prisma and sequel https://www.prisma.io/docs/getting-started/quickstart
